# picosh

"picosh" (pico shell) is a very basic Unix shell written in C.

## Installation

Clone the repo:

```bash
git clone https://github.com/bigkahuna00/picosh
```

Use the makefile:

```bash
make
```
Or compile using gcc:
```bash
gcc picosh.c -o picosh
```

## Usage

```bash
./picosh
```
